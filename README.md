# 酒店管理系统项目说明

## 项目简介
本项目是一个基于C++实现的简易酒店管理系统，适合大一学生学习和实践C++基础语法、结构体、链表、文件操作等知识点。系统支持客房和客人信息的增删查改、房间预订与退房、数据持久化等功能，采用Google C++风格编写，注重代码可读性和规范性。

## 主要功能
- **客房管理**：
  - 添加、查询、显示所有房间
  - 支持不同类型（如总统套房、双人房等）、价格、状态管理
  - 预订房间、退房
- **客人管理**：
  - 添加、查询、删除、修改客人信息
  - 支持身份证号唯一性校验
  - 记录入住、离开时间、房间编号、总花费
- **数据持久化**：
  - 所有数据均存储于`room.txt`和`guest.txt`，程序重启后数据不丢失
- **双层UI菜单**：
  - 一级菜单：客房相关、客人相关、退出系统
  - 二级菜单：各自类型的增删查改及返回主页面

## 文件结构
- `system.cpp`：主程序，包含所有功能实现
- `room.txt`：房间信息数据文件
- `guest.txt`：客人信息数据文件
- `README.md`：项目说明文档

## 编译与运行
1. 使用支持C++11及以上标准的编译器（如g++）
2. 在`project`目录下执行：
   ```sh
   g++ system.cpp -o system
   ./system
   ```
